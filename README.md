## 🚀 Flagship Projects

### 🔷 RACE — Production Layer-2 Blockchain Ecosystem
> *CDN Solutions Group — 2024*

A complete L2 chain and Real-World Asset platform built from scratch.

- **L2 Chain (OP-Stack)** — Production sequencer with custom Raft/Etcd failover; zero-downtime block production under node failure
- **Cross-Chain Bridge** — Bidirectional L1↔L2 bridge for native currency, ERC-20, ERC-721
- **RWA Platform (ERC-3643)** — Tokenization of Art, Gold, Watches & SPVs with Senior/Junior debt tranches, on-chain DAO governance, and compliance enforcement
- **MPC Wallet** — Alchemy Smart Accounts + DFNS MPC for gasless, keyless transactions

`OP-Stack` `Geth` `Etcd/Raft` `ERC-3643` `ERC-4337` `DFNS MPC` `Blockscout` `Proxyd` `AWS`

---

### 🟡 KnightSwap — DeFi Exchange & Yield Protocol
> *Spearmint Technologies — 2022–2024*

Full-stack DeFi protocol on BSC securing **$1M+ TVL**.

- **DEX** — AMM with optimized critical-path execution (~40% overhead reduction), liquidity mining
- **KnightCompounder** — ERC-4626 auto-compounding vault with cross-protocol yield aggregation
- **Multichain Launchpad** — Real-time WebSocket deposit engine; EIP-712 off-chain signatures cutting gas by **90%**
- **CipherCore** — zk-SNARK privacy mixer (Merkle Trees + nullifier sets)

`Solidity` `ERC-4626` `AMM` `zk-SNARKs` `BSC` `WebSockets` `EIP-712`

---

### 🟣 BaaS Orchestration Platform

Cloud-native platform for instant provisioning of private blockchain networks — functionally similar to a mini EKS control plane.

- Node.js orchestration API for Hyperledger Fabric & Quorum topologies
- Terraform-driven lifecycle management (provision → configure → teardown)
- Dynamic Bash templating engine for MSPs, CAs, Genesis blocks, Dockerfiles at runtime

`Node.js` `Terraform` `Hyperledger Fabric` `Quorum` `AWS` `Docker` `Bash`
